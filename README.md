
# Library Management System

Fully functional console-based library system to manage book inventory, issuing, returning, and category browsing.

## Features
- Add and manage books (ID, title, author, category)
- Issue/return books with validation
- Search books by ID
- Category-wise book display
- Menu-driven CLI with clean formatting

## Technologies Used
- C++ with OOP, Vectors
- Encapsulation & Modular Functions

## How to Run
1. Open terminal
2. Compile: `g++ main.cpp -o LibrarySystem`
3. Run: `./LibrarySystem`
